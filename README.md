Lost & Found App (SIT305 Task 7.1P)

Overview:
The app allows users to create and manage lost and found advertisements in a simple and user-friendly interface.

Users can:
Create lost or found item adverts,
View all saved items,
Filter items by category,
View detailed item information,
Remove items from the system,

Features:
Create New Advert
Select Lost or Found
Enter:
Name
Phone number
Description
Date & time
Location
Choose a category
Upload an image
Save advert to local database

View All Items:
Displays all saved adverts,
Each item shows:
Title (Lost/Found + Name),
Location,
Date,
Filter by Category,
Dropdown filter (Spinner),
Categories include:
Electronics,
Pets,
Wallets,
Keys,
Bags,
Documents,
Other,

Item Detail Screen:
Displays full information,
Title,
Date,
Location,
Description,
Phone number,
Category,
Timestamp,
Displays uploaded image,
Includes Remove button

Remove Item:
Deletes selected item,
Automatically updates list

Technologies Used:
Android Studio,
Room Database (SQLite),
XML UI Layouts,
Intents & Activities,
Spinner (Filtering)

How to Run:
Open project in Android Studio,
Sync Gradle
Run on:
Emulator OR
Physical Android device,
Image Upload (Important for Emulator)
If using emulator:
Drag & drop image into emulator
OR
Download an image using Chrome inside emulator
Then:
Tap Choose Image
Select the image
Save advert

Key Concepts Demonstrated:
Activity navigation using Intents,
Data persistence using Room Database,
UI development using XML,
Filtering using Spinner,
Custom ListView adapter,
User input validation
